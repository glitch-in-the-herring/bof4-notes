The general table for spells can be found at 
* `/BIN/SYSTEM/GAME.EMI: 0000ddd0`in the files
* `0x8019add0` in the RAM
starting with the empty spell. Each entry in the table is 12 bytes long and can be broken down as such:

| Position | Description                           | Value(s)                                                                                                                                                                                                            | Note                                                                                          |
| -------- | ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| 0        | Targeting                             | See [[Spells/Targeting\|Targeting]]                                                                                                                                                                                 |                                                                                               |
| 1        |                                       |                                                                                                                                                                                                                     |                                                                                               |
| 2        | Learnability and secondary resistance | First digit: Offset of the resistance. See [[Resistance]].<br><br>Second digt:<br>`0x_1` - `0x_8`: The spell is learnable. The second digit represents the difficulty of learning the spell. See also [[Learning]]. |                                                                                               |
| 3        | Power                                 | 8-bit unsigned integer                                                                                                                                                                                              | Not all spells use this                                                                       |
| 4        | AP cost                               | 8-bit unsigned integer                                                                                                                                                                                              |                                                                                               |
| 5        |                                       |                                                                                                                                                                                                                     |                                                                                               |
| 6        | Element                               | See [[Elements]]                                                                                                                                                                                                    | In some cases, is used to determine the primary resistances, notably by the elemental spells. |
| 7        |                                       |                                                                                                                                                                                                                     |                                                                                               |
| 8-9      | Icon/Type                             | See [[Spell Icons]]                                                                                                                                                                                                 |                                                                                               |
| 10-11    | Description Index                     | 16-bit unsigned integer                                                                                                                                                                                             |                                                                                               |
