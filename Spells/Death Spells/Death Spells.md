Death spells are spells that cause immediate death to the target, if it doesn't miss. While the name suggests that all death spells take the death resistance into account, this is not the case, as Kyrie takes into account the holy resistance instead.
* [[Spells/Death Spells/Kyrie]]
* [[Death]]

HP drop:
```
801d8f84 move    $v1, $r0 ; load 0
801d8f88 lw      $v0, -0x1ec4(v0) ; load the starting address of the target's battle section
801d8f8c nop     
801d8f90 sw      $v1, 0x00b0(v0) ; set the target's HP to zero
```
The above code can be found at `BIN/BATTLE/BTLMOVE.EMI: 00010f84`
Death status:

```
801b7078 li      $v0, 0x8000 ; load 0x8000 = disabled enemy
801b707c sh      $v0, 0x00ae(s0) ; store the new status
...
801b6df8 lhu     $v0, 0x00ae(s0) ; load current status
801b6dfc j       0x801b6e10
801b6e00 ori     $v0, 0x4000 ; combine with 0x4000 = enemy is hidden
801b6e10 sh      $v0, 0x00ae(s0) ; store the new status
```
