At the start of a battle, the [[Stat Multiplier|stat multipliers]] are all set to 100.
## Lower Limit
The lower limit is normally 50. This is determined by the following lines of code:
```
slti    $v0, $v1, 0x0032 ; Check if the current multiplier is less than 50
bne     $v0, $r0, 0xb01b1240
li      $v1, 0x0032 ; If it is, then cap it to 50
```

## Upper Limit
The upper limit is normally 200. Because MIPS32 only supports less than comparisons, the multiplier is actually checked against 200 + 1 = 201 instead.
```
slti    $v0, $v1, 0x00c9 ; Check if the current multiplier is less than 201
bne     $v0, $r0, 0x801b1224
li      $v0, 0x00c8 ; If it is not, then cap it to 200
```
