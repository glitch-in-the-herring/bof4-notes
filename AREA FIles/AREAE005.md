## Text
The small amount of non pre-rendered text that is used on this .AREA file can be found at `0x93010` in the file. The location name popups are pre-rendered and cannot be changed by changing the strings in this section
## Buttons
* The button that is used to add speed to the shift selector can be found at
	* `0x801f27cc` in the RAM
	* `0x000b77cc` in the game's files
* The button that is used to camp can be found at
	* `0x801f19b8` in the RAM
	* `0x000b69b8` in the game's files
