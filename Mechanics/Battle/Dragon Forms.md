Using [[Meditate]], Ryu and Fou-Lu are able to transform into dragons.
## Order
The internal order used for the dragons are:
* Aura
* Kaiser
* Wyvern
* Weyr
* Astral
* Tyrant
* Serpent
* Peist
* Myrmidon
* Knight
* Behemoth
* Mammoth
* Mutant
* Punk

### Pre-Chapter 3
Before chapter 3, the first spell of all dragon forms are locked.
```
80191834 lbu     $v0, 0x0000(a0) ; load the chapter flag byte
80191838 andi    $a1, 0x0007 
8019183c srav    $v0, $a1 ; shift the bits
80191840 jr      $ra
80191844 andi    $v0, 0x0001 ; get the chapter flag
801d15f4 andi    $v0, 0x00ff
801d15f8 sltiu   $a1, $v0, 0x0001 ; if the chapter flag is not set, then set the shift to 1
```
