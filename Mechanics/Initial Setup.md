When starting a new game, the game predefines some values
## Character data
Initial character data can be found at:
* `BIN/SYSTEM/GAME.EMI: 000203e0` in the files
* `0x801b53e0` in the RAM
See [[Character Data]] for the format
## Inventory
### Ryu
Ryu's initial inventory can be found at:
* `BIN/SYSTEM/GAME.EMI: 0002083c` in the files
* `0x801b583c` in the RAM
