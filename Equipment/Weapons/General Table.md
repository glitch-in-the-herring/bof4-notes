The general table for weapons can be found at:
* `BIN/SYSTEM/GAME.EMI: 0000d154` in the files
* `0x8019a154` in the RAM
Starting with the empty weapon. Each entry is 14 bytes long and is structured as follows:

| Position | Description        | Value(s)                                                                                                                                                         | Note                                                                            |
| -------- | ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| 0        |                    |                                                                                                                                                                  |                                                                                 |
| 1        | Flags              | See ...                                                                                                                                                          |                                                                                 |
| 2        | Icon/Type          | `0x0_`: Sword<br>`0x1_`: Sword?<br>`0x2_`: Club<br>`0x3_`: Staff<br>`0x4_`: Gun<br>`0x5_`: Multi-target gun?<br>`0x6_`: Knuckles                                 | `0x1_` weapons are exclusively used by Scias.                                   |
| 3        |                    |                                                                                                                                                                  |                                                                                 |
| 4        | Equipability       | See [[Equipability]]                                                                                                                                             |                                                                                 |
| 5        | Element            | See [[Elements]]                                                                                                                                                 |                                                                                 |
| 6        | Weight             | 8-bit unsigned integer                                                                                                                                           |                                                                                 |
| 7        | Hits/Shield Damage | The first digit of this byte determines the number of hits the weapon has. The second determines how much shield points does the weapon take out when attacking. |                                                                                 |
| 8-9      | Power              | 16-bit unsigned integer                                                                                                                                          |                                                                                 |
| 10-11    | Description Index  | 16-bit integer                                                                                                                                                   |                                                                                 |
| 12-13    | Selling price      | 16-bit integer                                                                                                                                                   | The purchasing price is automatically calculated by doubling the selling price. |
